# 💃 Dance Academy Management System

A modern and responsive Dance Academy Management website developed using **HTML**, **Tailwind CSS**, and **JavaScript**. This project provides a user-friendly interface for students to explore dance programs, register for admissions, and learn about the academy.

---

## 📌 Features

- 🏠 Home Page
- 👨‍🏫 About Us
- 💃 Dance Styles
- 📝 Admission Form
- 📞 Contact Us
- 🖼️ Gallery
- 🎉 Events
- 👩‍🏫 Trainers
- ❓ Frequently Asked Questions (FAQ)
- 🔐 Login Page
- ✍️ Sign Up Page
- 🔑 Forgot Password
- 📄 Privacy Policy
- 📃 Terms & Conditions
- 📩 Enquiry Form

---

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- JavaScript
